Lens Proxy Work in progress

2026
Golang · TCP / wire protocols · SQL AST parsing · proxy re-encryption

Lens is a database proxy engineered to automate data privacy and compliance by dynamically redacting and masking sensitive information in real time. It audits every query sent to the database and supports query blocking, temporary or one-time access, and a review workflow prior to execution — effectively turning the database boundary into a programmable policy plane.

Goval

2023
Golang · functional composition · zero-reflection

A high-performance, zero-reflection validation library for Go — built around type safety and developer ergonomics. It leans on functional composition for declarative rules while sidestepping reflect overhead, so it stays fast on the hot path. Concurrency-safe, easily extensible, and built for production use.

github.com/pkg-id/goval

Gomap

2023
Golang · generics · declarative API

A type-safe utility library that uses Go generics to give maps a declarative API — high-performance primitives for filtering, reducing, and mapping values, plus set operations like intersection and difference. Designed for extensibility, with the goal of expressing common transformations in as little boilerplate as possible.

github.com/dimasadyaksa/gomap

OptiScale

2024
Internal at Traveloka · Golang · AWS APIs · rule engine

An automated advisory platform that reads AWS service metrics and resource configurations, then evaluates them against industry best practices and internal conventions. Internal · not open source
